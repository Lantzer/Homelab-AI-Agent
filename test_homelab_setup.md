# Local Homelab
## Hardware
Raspberry Pi 5 with PoE+ Hat and 500 gb nvme ssd for storage
Xfinity Modem in bridge mode
Archer c7 router
POE Switch

Physical setup: Xfinity Modem to Router from there it goes to a hue hub and to a switch the switch is connected to the raspberry pi via poe+ serving it power and ethernet

## Raspberry Pi
Raspberry Pi OS
Running Docker
Docker containers running: 
- PiHole, adblock
- HomePage, local to display services
- Mealie, local meal website
